四大核心工具

覆盖日常记录、文案、计算与预算管理，专注实用与效率。

工时统计

从日志月历页读取填报情况，统计标准工时、实填工时、工时差额和未填工作日。

  • 识别工作日、周末、休假与补班
  • 保存单月月报并生成历史列表
  • 下载单月或汇总 Excel 表格

快速文案

沉淀常用文本和回复模板，支持搜索、分组、一键复制与填入当前编辑区域。

  • 按分组管理常用文案
  • 支持搜索与快速筛选
  • 兼容输入框、文本域和 iframe 场景

工时计算器

在设计任务页面读取任务名称与预计工时，保存后用于跟踪任务填写进度。

  • 自动扫描设计任务列表
  • 支持选中保存和重复识别
  • 按小时、人天规则汇总结果

预算计算器

读取任务详情中的项目、任务、总额与日志明细，按月份沉淀预算记录。

  • 常驻保存整月日志
  • 识别跨月任务并按当月工时分摊
  • 汇总已保存预算并提示异常值

插件使用手册

按工具面板左侧四个功能整理，进入对应业务页面后打开面板即可使用。

工时统计

用于从日志月历页读取当月填报情况，并沉淀月报记录。

使用步骤
  1. 进入新点 OA 的日志月历页，确认页面已经显示目标月份。
  2. 打开插件面板，切换到「工时统计」。
  3. 查看标准工作天数、实际填工时天数、标准工时、实际工时和工时差额。
  4. 确认数据无误后点击「保存月报」。
  5. 在「已保存月报」中查看历史月份，按需下载单月或下载总表。
注意事项
  • 未进入日志月历页时，插件会提示进入对应页面后再读取。
  • 月报会保存到当前浏览器本地存储，更换浏览器或清理数据后需重新保存。
  • 遇到请假、补班、法定休等日期，请以统计卡片中的分类结果为准再核对。

快速文案

用于管理常用回复、日报片段、确认话术，并快速复制或填入输入框。

使用步骤
  1. 切换到「快速文案」，点击「新建」。
  2. 选择已有分组，或在「新建分组」中输入新的分组名称。
  3. 填写文案内容后点击「保存文案」。
  4. 通过搜索框或分组筛选定位文案。
  5. 点击发送图标快速填入当前编辑区域，或点击复制图标复制到剪贴板。
管理方式
  • 分组支持折叠、改名和删除，默认分组不可直接删除。
  • 单条文案支持编辑和删除。
  • 当前页面找不到可填入输入框时，插件会退回为复制文本。

工时计算器

用于读取设计任务预计工时，保存后跟踪已填工时和完成比例。

使用步骤
  1. 进入包含设计任务列表或任务信息的页面。
  2. 切换到「工时计算器」，点击「重新读取」。
  3. 在「读取到的设计任务」中勾选需要跟踪的任务。
  4. 点击「保存选中」，任务会进入「已保存任务」。
  5. 在已保存任务中填写「已填」工时，查看进度条和完成百分比。
核对规则
  • 已保存过的任务会显示为「已保存」，避免重复保存。
  • 已填工时超过预计工时时，百分比会继续显示，便于发现超工时。
  • 读取不到任务时，先确认当前页面是否包含任务名称、设计师和预计工时。

预算计算器

用于读取任务预算和日志明细，按月份保存预算并汇总金额。

使用步骤
  1. 先打开整月日志页，让插件读取当月日志，并在「常驻月日志」中点击「保存常驻」。
  2. 进入任务详情页，切换到「预算计算器」。
  3. 确认项目名称、任务名称和预算金额已经读取出来。
  4. 点击当前任务卡片中的「保存」。
  5. 在「已保存预算」中查看当月预算列表和合计金额。
特殊情况
  • 预算为 0 时会弹出确认，建议先核对任务预算信息是否遗漏。
  • 跨月任务会按当月工时占总工时的比例分摊预算，保存前会显示确认信息。
  • 常驻月日志可按年月切换，也可以删除不再需要的月份记录。

所有月报、文案、任务和预算记录默认保存到当前 Chrome 浏览器本地存储中；公开网页仅提供说明与下载，不会上传你的业务数据。

插件安装方法

以谷歌浏览器和 Microsoft Edge 为例，下载后先解压，再通过扩展管理页加载已解压的扩展程序。

谷歌浏览器安装

适用于 Google Chrome、Chromium 内核浏览器。

  1. 点击页面上的「下载插件安装包」，下载插件压缩包。
  2. 右键解压到固定文件夹，保留解压后的完整目录。
  3. 在地址栏输入 chrome://extensions/ 并回车。
  4. 打开右上角「开发者模式」。
  5. 点击「加载已解压的扩展程序」，选择包含 manifest.json 的解压文件夹。
  6. 安装完成后，在扩展图标菜单中固定「新点小工具」，方便日常打开。

Microsoft Edge 安装

适用于 Windows 默认 Edge 浏览器。

  1. 下载同一个插件安装包，并先解压到固定文件夹。
  2. 在地址栏输入 edge://extensions/ 并回车。
  3. 打开左侧或页面中的「开发人员模式」。
  4. 点击「加载解压缩的扩展」，选择包含 manifest.json 的文件夹。
  5. 安装完成后，在工具栏扩展菜单里固定「新点小工具」。
  6. 更新版本时，用新文件覆盖解压目录，再到扩展页点击「重新加载」。
先解压：不要直接选择 zip 文件，浏览器只识别解压后的扩展文件夹。 别移动：安装后不要随意删除或移动解压目录，否则扩展会失效。 来源提示：如浏览器弹出安全确认，选择保留或允许后继续安装。
